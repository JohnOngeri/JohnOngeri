### 🔐 PriFed — Privacy-Preserving Federated Learning

The hardest engineering problem I've tackled. Hospitals and clinics can't share patient records to train a shared model — the privacy risk is too high. I built **PriFed** to eliminate that tradeoff: multiple institutions train collaboratively using Federated Averaging, raw data never leaves any device, and Differential Privacy guarantees cap how much any gradient update can reveal.

This is the architecture that runs at the core of Google Health, Apple's on-device ML, and Meta's private inference pipelines.

### 🧠 NLP Sentiment Analysis — 4-Model Showdown

I benchmarked four NLP architectures on 50,000 IMDB reviews to answer a real engineering question: *does complexity actually pay off?* A well-tuned Logistic Regression hit **88% F1** at one-tenth the compute cost of a BiLSTM with Word2Vec. I wrote it up. The finding matters for anyone deciding whether to reach for a transformer at production scale.
